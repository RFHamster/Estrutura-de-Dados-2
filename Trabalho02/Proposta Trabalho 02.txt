Busca em texto – Boyer-Moore

Não compacte os arquivos.

 

Descrição do trabalho

 

Criar um programa que:

- peça ao usuário o nome do arquivo de texto a ser trabalhado,

- mostre o texto lido,

- iterativamente, pedir ao usuário uma cadeia de caracteres a ser buscada no texto e uma cadeia de caracteres que deve substituir toda aparição da primeira cadeia,

- use o algoritmo de Boyer-Moore para fazer as buscas,

- reapresentar o texto alterado

- terminar a iteração quando o usuário inserir uma cadeia vazia,

- pedir ao usuário um nome para salvar a versão final.

 

Implementação

 

O problema deve ser resolvido usando os algoritmos dados em aula.

O programa deve possuir comentários suficientes para que se compreenda seus passos.
